# gopeed-extension-pixiv

a gopeed extension for downloading pixiv images.

## supported urls

```
https://www.pixiv.net/en/artworks/123456789
https://www.pixiv.net/artworks/123456789
https://pixiv.net/en/artworks/123456789
```

## installation

1. download the latest release or build from source
2. open gopeed settings > extensions
3. install from local file or drag and drop the extension

### building from source

```bash
npm install
npm run build
```

the built extension will be in `dist/index.js`.
